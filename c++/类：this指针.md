所有非静态成员函数都有一个名为 `this` 的 const 指针，它保存隐式对象的地址：
- `static` 关键字表示该函数不与类的对象关联，而是被视为位于类作用域内的普通函数：涉及：**静态成员函数**
- 对于非常量成员函数， `this` 是一个指向非常量值的常量指针（意味着 `this` 不能指向其他对象，但所指向的对象可以被修改）。对于常量成员函数， `this` 是一个指向常量值的常量指针（意味着指针不能指向其他对象，且所指向的对象也不能被修改）。

让成员函数将隐式对象作为返回值返回会很有用。这样做的主要目的是允许将成员函数“链式”调用，从而可以在单个表达式中对同一对象连续调用多个成员函数！这称为**函数链式调用**（或**方法链式调用**）。

将类重置为默认状态的最佳方法是创建一个 `reset()` 成员函数，让该函数创建一个新对象（使用默认构造函数），然后将这个新对象赋值给当前的隐式对象，如下所示：

```cpp
void reset()
{
    *this = {}; // value initialize a new object and overwrite the implicit object
}
```
