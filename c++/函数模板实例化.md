
```cpp
#include <iostream>

template <typename T>
T max(T x, T y)
{
    return (x < y) ? y : x;
}

int main()
{
    std::cout << max<int>(1, 2) << '\n'; // instantiates and calls function max<int>(int, int)

    return 0;
}
```

从**函数模板**（带有模板类型）创建（具有特定类型的）函数的过程称为**函数模板实例化**（简称实例化）。
- 当函数因函数调用而被实例化时，这称为**隐式实例化**。
- 从模板实例化而来的函数在技术上称为**特化**，但在日常用语中通常称为**函数实例**。
产生特化的模板称为**主模板**。函数实例在所有方面都是普通函数


在参数类型与我们想要的实际类型匹配的情况下，我们不需要指定实际类型
- 我们可以使用**模板参数推导**，让编译器从函数调用中的参数类型推导出应该使用的实际类型。
```cpp
std::cout << max<>(1, 2) << '\n';
std::cout << max(1, 2) << '\n';
```
> 普通函数调用语法会优先选择非模板函数，而非从模板实例化出的同等可行的函数。

> 在调用从函数模板实例化的函数时，优先使用普通函数调用语法（除非需要函数模板版本优先于匹配的非模板函数）。
