当函数被调用时，其所有**形参**都会作为变量被创建，并且每个**实参**的值会被复制到对应的形参中（使用**复制初始化**）。这个过程称为**按值传递**。采用按值传递的函数形参被称为**值参数**。


在某些情况下，你会遇到函数参数在函数体内未被使用的情况。这些参数被称为**未引用参数**。

在函数定义中，函数参数的名称是可选的。
- 当函数参数需要存在但未在函数体中使用时，可以省略其名称。
- 没有名称的参数被称为**未命名参数**
```cpp
void doSomething(int) // ok: unnamed parameter will not generate warning
{
}
void doSomething(int /*count*/)
{
}
```

带有参数却又不使用其值的函数。这种情况最常见于以下类似场景
- 该函数以某种方式进行了更新，参数的值不再需要
- 运算符 `++` 和 `--` 具有前缀和后缀变体（未引用的函数参数用于区分此类运算符的重载是针对前缀还是后缀情况)：涉及重载递增递减运算符
- 需要从类型模板参数的类型（而非其值）中确定某些信息时：涉及类型模板
