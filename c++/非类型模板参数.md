**非类型模板参数**是一种具有固定类型的模板参数，它充当作为模板参数传入的 **constexpr** 值的占位符。
- 函数模板针对不同类型的非类型模板参数进行了重载，很容易导致匹配不明确


非类型模板参数可以是以下任意类型：
- 整型
-  枚举类型
- `std::nullptr_t`
- 浮点类型（自 C++20 起）
- 指向对象的指针或引用
- 指向函数的指针或引用
- 指向成员函数的指针或引用
- 字面量类类型（自 C++20 起）

涉及：`std::bitset`

```cpp
#include <bitset>

int main()
{
    std::bitset<8> bits{ 0b0000'0101 }; // The <8> is a non-type template parameter

    return 0;
}
```
这是一个使用整型非类型模板参数的简单函数示例：
```cpp
#include <iostream>

template <int N> // declare a non-type template parameter of type int named N
void print()
{
    std::cout << N << '\n'; // use value of N here
}

int main()
{
    print<5>(); // 5 is our non-type template argument

    return 0;
}
```

> 非类型模板参数主要用于需要将常量表达式值传递给函数（或类类型）的情况，以便它们可以在需要常量表达式的上下文中使用。
> 类类型 `std::bitset` 使用非类型模板参数来定义存储的位数，因为位数必须是一个常量表达式值。

某些非类型模板参数可以隐式转换，以匹配不同类型的非类型模板参数
```cpp
#include <iostream>

template <int N> // int non-type template parameter
void print()
{
    std::cout << N << '\n';
}

int main()
{
    print<5>();   // no conversion necessary
    print<'c'>(); // 'c' converted to type int, prints 99

    return 0;
}
```
仅允许特定类型的 constexpr 转换。最常见的允许转换类型包括：
- 整型提升
- 整型转换
- 用户定义转换
- 左值到右值的转换


从 C++17 开始，非类型模板参数可以使用 `auto` 让编译器从模板参数推导出非类型模板参数：
```cpp
#include <iostream>

template <auto N> // deduce non-type template parameter from template argument
void print()
{
    std::cout << N << '\n';
}

int main()
{
    print<5>();   // N deduced as int `5`
    print<'c'>(); // N deduced as char `c`

    return 0;
}
```
