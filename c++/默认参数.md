如果一个参数被赋予了默认实参，那么其后的所有参数（右侧）也必须被赋予默认实参。

一旦声明，默认实参在同一翻译单元内不能被重新声明。这意味着对于有前向声明和函数定义的函数，默认实参可以在前向声明或函数定义中声明，**但不能两者都声明**。

如果函数有**前向声明**（尤其是在头文件中），请将默认参数放在那里。否则，将默认参数放在函数定义中。

默认值不属于函数签名的一部分，因此这些函数声明属于不同的重载。
```cpp
void print(int x);                  // signature print(int)
void print(int x, int y = 10);      // signature print(int, int)
void print(int x, double y = 20.5); // signature print(int, double)
```

默认参数很容易导致函数调用产生歧义：
```cpp
void foo(int x = 0)
{
}

void foo(double d = 0.0)
{
}

int main()
{
    foo(); // ambiguous function call

    return 0;
}
```

默认参数不适用于通过函数指针调用的函数：涉及：**函数指针**
