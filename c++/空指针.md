除了内存地址，指针还可以存储一个额外的值：**空值**。空值（通常简称为 null）是一个特殊值，表示某个值不存在。当指针存储空值时，意味着该指针不指向任何内容。这种指针称为**空指针**。
- 如果你没有用有效对象的地址初始化指针，请使用值初始化（使其成为空指针）。
- **nullptr** 关键字代表一个空指针字面量。我们可以使用 `nullptr` 来显式地初始化或为指针赋一个 null 值。
- 传统的空指针字面量：0 和 NULL
- 解引用空指针也会导致未定义行为


> 指针要么持有有效对象的地址，要么被设置为 nullptr。这样我们只需要测试指针是否为空，并可以假设任何非空指针都是有效的

在函数重载中， `nullptr` 可以与整数值区分开来，因此它必须具有不同的类型。那么 `nullptr` 的类型是什么呢？
- 答案是 `nullptr` 的类型为 `std::nullptr_t` （定义在头文件中）。 `std::nullptr_t` 只能容纳一个值： `nullptr`
