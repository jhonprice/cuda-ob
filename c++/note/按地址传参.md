通过按**地址传递**，调用者不是提供对象作为参数，而是提供对象的地址（通过指针）。
- 该指针（持有对象地址）被复制到被调用函数的指针参数中（现在也持有对象地址）。
- 函数可以通过解引用该指针来访问传递的地址所对应的对象。


除非函数需要修改传入的对象，否则应优先使用指向常量的函数参数，而非指向非常量的函数参数。
- 除非有特殊原因，否则不要将函数参数定义为 const 指针


> 除非有特殊原因需要使用按地址传递，否则应优先选择按引用传递。

示例如下:
```cpp 
#include <iostream>

// foo函数声明

void foo(const std::string* str);

  

int main() {

    std::cout << "Hello, World!" << std::endl;

    std::string str = "This is foo function pass by address";

    foo(&str);

    return 0;

}

// foo函数定义

void foo(const std::string* str) {

    std::cout << *str << std::endl;

}
```
