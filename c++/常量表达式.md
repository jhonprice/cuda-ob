根据 **as-if 规则**，编译器可以选择在运行时或编译时求值某些表达式：涉及**Constexpr变量**，**非类型模板参数**，**数组**

一些需要编译时可求值表达式的常见情况
- constexpr 变量的初始化器
- 非类型模板参数
- `std::array` 或 C 风格数组的定义长度。

使用导致编译时求值的语言特性被称为**编译时编程**，涉及
- Constexpr 变量
- 常量表达式函数
- 模板
- 静态断言
所有这些功能都有一个共同点：它们都利用了常量表达式。

**常量表达式**则是一个非空的字面量、常量变量、运算符和函数调用的序列，*所有这些都必须在编译时可求值*

最常见的常量表达式包含以下内容：
- 字面量
- 操作符带有常量表达式操作数（例如 `3 + 4` 、 `2 * sizeof(int)` ）。  
- 具有常量表达式初始化器的常量整型变量（例如 `const int x { 5 };` ）。
-  Constexpr 变量
- 带有常量表达式参数的 constexpr 函数调用，涉及：Constexpr 函数

> 常量表达式也可以包含
> - 非类型模板参数
> - 枚举项
> - 类型特征
> - 常量表达式 lambda
