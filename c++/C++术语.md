C++中有许多不同类型的**语句**：
- Declaration statements  声明语句
- Jump statements  跳转语句
- Expression statements  表达式语句
- Compound statements  复合语句
- Selection statements (conditionals)  条件语句
- Iteration statements (loops)  循环语句
- Try blocks  Try 块

通常，**注释**应用于三个方面。

1. 首先，对于给定的库、程序或函数，注释最适合用来描述该库、程序或函数的功能。这类注释通常放置在文件或库的顶部，或紧邻函数之前
2. 其次，在上述描述的库、程序或函数内部，注释可用于说明代码将如何实现其目标
3. 第三，在语句层面，注释应当用于解释代码为何执行某项操作。
   - 糟糕的语句注释会描述代码在做什么。
   - 如果你编写的代码复杂到需要注释来解释语句的功能，那么你可能需要重写该语句，而非添加注释。

**数据**指的是任何可以被计算机移动、处理或存储的信息

**对象**用于在内存中存储值（对象就是内存区域）。变量是具有名称（标识符）的对象。  
- 为对象命名使我们能够在程序后续部分再次引用这些对象。
- 在一般编程中，术语“对象”通常指内存中的未命名对象、变量或函数。在 C++中，术语“对象”的定义更为狭窄，排除了函数
- 对象在为其使用预留了实际存储空间时即被“创建”。
- 对象的类型必须在编译时确定（这样编译器才能知道该对象需要多少内存）

**定义**语句可用于告知编译器我们希望在程序中使用一个变量。

**字面量**是直接插入源代码中的值。这些值通常会直接出现在可执行代码中（除非被优化掉）。

**运算**是一个涉及零个或多个输入值（称为**操作数**）并产生新值（称为*输出值*）的过程。要执行的具体运算由一个称为**运算符**的符号表示。

C++中的运算符分为四种不同的元数
- **零元运算符**作用于零个操作数。在 C++中也只有一种这样的运算符（throw 运算符）

运算符的执行顺序:
- 先括号，然后指数，接着乘除，最后加减。
- 这种顺序有时缩写为 PEMDAS，或扩展为助记短语"Please Excuse My Dear Aunt Sally"。

在 C++中，“**副作用**”一词具有不同的含义：它指的是运算符或函数除了产生**返回值**之外，还能产生可观察的效果。

**表达式**是由字面量、变量、运算符和函数调用组成的非空序列，用于计算一个值。执行表达式的过程称为**求值**，而计算出的值称为表达式的**结果**（有时也称为**返回值**）。
- 字面量会求值为其自身的值。
- 变量会求值为该变量的值。
- 运算符（例如 `operator+` ）会使用其操作数来求值为其他值。
- 函数调用会求值为该函数返回的任何值。
```cpp
x = 5           // x = 5 的副作用是将 5 赋值给 x，表达式求值结果为 x
x = 2 + 3       // 副作用是将 5 赋值给 x，表达式求值结果为 x
std::cout << x  // 副作用是将 x 的值打印到控制台，表达式求值结果为 std::cout
```


在 C++中，表达式的**结果**是以下之一：
- 一个值（最常见的情况）
- 一个对象或函数:涉及 左值和右值
- 无值。这些是非值返回函数调用的结果

"**foo**"是一个无意义的词，常被用作函数或变量的占位符名称，当名称对于展示某些概念并不重要时。这类词被称为元语法变量（尽管在日常语言中，它们常被称为"占位符名称"，因为没人能记住"元语法变量"这个术语）。C++中其他常见的元语法变量包括"bar"、"baz"，以及以"oo"结尾的三个字母单词，如"goo"、"moo"和"boo"。
